# Viv Calif Sales Customer Dashboard (Excel)
## Project Overview
This project showcases an interactive E-commerce Sales Dashboard built using Microsoft Excel with Pivot Tables and Charts.

The dashboard provides deep insights into sales performance, customer behavior, product popularity, shipping efficiency, and customer satisfaction.

It is designed to help businesses make data-driven decisions by analyzing key metrics in a simple and visual way.


## Objectives
The main objectives of this project are:

• Track overall sales performance

• Analyze order quantity and revenue trends

• Understand customer purchase behavior

• Identify popular products

• Evaluate delivery performance

• Measure customer satisfaction


## Dashboard Features
The dashboard includes the following key visualizations:

• Sales & Orders Analysis

• Total Orders, Quantity, and Revenue KPIs

• Weekly trend of Quantity & Sales Amount (Last 13 weeks)

• Customer Behavior Insights

• Purchase channels (App, Website, Instagram, etc.)

• Buying frequency distribution

• Gender-based purchase behavior

• Product Analysis

• Most popular products by category

• Gender-wise product preference

• Geographical Analysis

• Customer distribution by location (map visualization)

• Shipping & Delivery Analysis

• Delivery time distribution

• Average days to deliver

• Customer Satisfaction

• Monthly customer satisfaction ratings


## Filters / Slicers
Interactive slicers are used to enhance user experience:

• Order Mode (App, Website, Instagram, etc.)

• Gender (Male, Female, Other, Unknown)

These filters allow dynamic and customized analysis.


## Key Insights
• Female customers contribute more to total purchases

• Most customers buy products 2–3 times

• T-Shirts, Jeans, and Sneakers are top-selling products

• App and Website are the most preferred purchase channels

• Average delivery time is around 2–3 days

• Customer satisfaction ratings are generally high
